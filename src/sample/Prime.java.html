@param size_limit the length, in digits, that the class should be allowed to calculate. If you set the limit lower, it will compute primes faster, but they cannot be as long.     private boolean setSizeLimit(int size_limit){
          boolean settable = false;
          if (size_limit < 40){
              settable = true;
              this.sizeLimit = size_limit;
          }

          return settable;

    private boolean isPrime(int integer){
        boolean isPrime = true;
        for (int i = 2; i < integer; i++){ 
            if (integer % i == 0){
                isPrime = false;
